## 📚 学习路径

### 新手入门

```
特征工程基础 → LightGBM原理 → 模型训练 → 投资组合构建 → 策略回测
```

适合初学者，从基础概念开始，逐步学习完整的量化投资流程。

### 进阶提升

```
时序数据划分 → IC优化训练 → 特征重要性分析 → 成本模型 → 绩效评估
```

适合有一定基础的学习者，深入理解和优化各个环节。

### 实战应用

```
从实际项目出发 → 遇到问题查文档 → 理论学习 → 实践应用
```

适合有经验的开发者，通过解决实际问题提升技能。

## 💡 使用技巧

### 如何使用本站

1. **按学习路径学习**：从Week 1开始，系统学习量化投资知识
2. **按主题查找**：使用标签索引快速找到感兴趣的内容
3. **理论与实践结合**：每个模块都包含代码示例，建议动手实践
4. **遇到问题查阅**：将本站作为参考文档，随时查阅相关知识点

### 浏览建议

- 首次访问建议阅读[快速开始](快速开始.md)
- 按照推荐的学习路径逐步学习
- 动手运行代码示例加深理解
- 遇到问题使用搜索功能或查阅站点导航
