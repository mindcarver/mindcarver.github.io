基本结构

训练循环流程：

外层循环 (Epochs)
    │
    ├──→ 遍历批次 (Batches)
    │       │
    │       ├──→ 前向传播: y_pred = model(x)
    │       │
    │       ├──→ 计算损失: loss = criterion(y_pred, y_true)
    │       │
    │       ├──→ 梯度清零: optimizer.zero_grad()
    │       │
    │       ├──→ 反向传播: loss.backward()
    │       │
    │       └──→ 参数更新: optimizer.step()
    │
    ├──→ 验证集评估
    │
    └──→ 保存最佳模型

优化器实现

优化器选择建议

选择决策树：

有充足调参时间?
├─ 是 → AdamW（最新实践）
│      ├─ Transformer 模型：weight_decay=0.01
│      └─ LSTM 模型：weight_decay=0.001
└─ 否 → Adam（稳妥选择）
       └─ lr=0.001, 默认参数

数据量特别大?
└─ 考虑 SGD + Momentum（训练更快）

需要强泛化能力?
└─ AdamW + 高 weight_decay
