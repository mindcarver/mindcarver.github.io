
# 回测引擎与策略实战

回测（Backtesting）是量化投资的核心环节，通过历史数据验证交易策略的有效性。本模块系统讲解了回测引擎、投资组合构建、绩效评估等关键内容。

## 🎯 学习路径

### 🟢 初学者路径

```
交易策略理论 → 投资组合构建 → 绩效评估 → 回测流程
```

**目标**：理解回测基本概念，能够完成简单策略回测

### 🟡 进阶路径

```
成本模型 → 参数敏感性 → 样本外验证 → 策略优化
```

**目标**：深入理解回测细节，能够优化和验证策略

### 🔴 实战路径

```
从实际项目出发 → 遇到问题查文档 → 理论学习 → 实践应用
```

**目标**：解决实际问题，积累实战经验

## 📚 学习前准备

### 必备知识

- [x] **Python 编程基础**
  - 数据处理（pandas, numpy）
  - 可视化（matplotlib, seaborn）
  
- [x] **机器学习基础**
  - LightGBM 模型训练
  - IC/ICIR 评估指标

- [x] **量化投资基础**
  - 特征工程（Qlib）
  - 时序数据划分

### 推荐顺序

1. 先学习 **特征工程模块** 和 **LightGBM 模块**
2. 再学习本回测模块
3. 最后进行实战练习
