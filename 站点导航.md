## 📚 学习路径

### 新手入门路径

```
特征工程基础 → LightGBM 原理 → 模型训练 → 投资组合构建 → 策略回测
```

**适合人群**：初学者，从基础概念开始，逐步学习完整的量化投资流程。

**建议时间**：4-6周

### 进阶提升路径

```
时序数据划分 → IC 优化训练 → 特征重要性分析 → 成本模型 → 参数优化
```

**适合人群**：有一定基础的学习者，深入理解和优化各个环节。

**建议时间**：6-8周

### 实战应用路径

```
从实际项目出发 → 遇到问题查文档 → 理论学习 → 实践应用
```

**适合人群**：有经验的开发者，通过解决实际问题提升技能。

**建议时间**：持续进行
