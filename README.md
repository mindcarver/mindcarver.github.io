# 量化投资技术文档

本仓库包含量化投资技术文档，部署在 GitHub Pages 上。

## 📚 内容

- **特征工程模块**：系统讲解 Qlib 特征工程的核心概念与实践方法
- **LightGBM 模块**：深入学习 LightGBM 在量化投资中的应用

## 🚀 快速开始

访问在线文档：[量化投资技术文档](/)

## 📖 本地开发

### 环境要求

- Ruby
- Bundler
- Jekyll

### 安装依赖

```bash
gem install jekyll
```

### 启动本地服务器

```bash
bundle exec jekyll serve
```

访问 http://localhost:4000 查看文档。
